# 贡献指南 (Contributing Guidelines)

感谢您参与《数字逻辑与数字电路》开源教材的编写！为了保证教材质量和协作顺畅，请遵循以下规则：

## 一、 如何做出贡献
1. [cite_start]**拥有账号**：确保您拥有一个 GitHub 账号 [cite: 2]。
2. [cite_start]**创建分支**：在 GitHub 上创建一个新分支（例如 `my_new_branch`）[cite: 2]。
3. **克隆与修改**：
   - [cite_start]使用 `git clone` 将存储库下载到本地 [cite: 2]。
   - [cite_start]更改分支：`git checkout -b my_new_branch` [cite: 2]。
   - [cite_start]进行修改后，使用 `git add`、`git commit` 和 `git push` 提交代码 [cite: 2]。
4. [cite_start]**提交申请**：通过 GitHub 提交 **Pull Request**，等待核心贡献者审核合并 [cite: 2]。

## 二、 编写准则
- [cite_start]**语言**：请尽量使用英语或规范的学术中文 [cite: 2]。
- [cite_start]**代码规范**：请检查现有的 Verilog 或仿真代码，确保编码模式一致 [cite: 2]。
- [cite_start]**LaTeX 公式**：请参考当前文件的 LaTeX 编写模式 [cite: 2]。
- [cite_start]**PDF 资源**：若需上传大体积 PDF 资源，请联系项目维护人员 [cite: 2]。
